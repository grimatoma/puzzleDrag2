Decision to make: "high top-down" (c1/c3) renders the soil as a clean raised isometric land-block — discrete and tile-like. "low top-down" (c2/c4) gives a flatter, more organic ground but is messier and over-matures the corn. The board grid stays a square 6×6 either way — this is tile art, not grid projection. If you'd rather have the flatter square soil-pad from your reference photos, say so and I'll reprompt for a thin flat pad instead of a raised block.
