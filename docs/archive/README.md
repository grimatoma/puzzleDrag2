# docs/archive

Outdated documentation kept for historical reference. Nothing here describes
the current design or active work — it has either **shipped** (so the plan/spec
is now just history) or been **superseded** by the live seasonal-tile pipeline.

Files keep their original relative `assets/…` image links, so they still render
when opened from inside this folder.

## What's here

### Implementation plans & specs (work shipped)
`superpowers/plans/` and `superpowers/specs/` — point-in-time plans and design
specs for features that are now implemented in `src/`:

- Appearance/look restructure
- Board-kinds wiki section
- Wiki interconnection IA
- Progression feed (phase 1) + engine migration (phase 2 / 2b)
- Phaser UX batch (2026-06-10)
