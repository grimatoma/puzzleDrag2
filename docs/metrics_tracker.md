# Progression Metrics — Zone One (Hearthwood Vale)

This document tracks the economic balancing and progression metrics for the initial game experience (Act 1).

## 1. Onboarding Metrics (Hearthwood Vale)

| Step | Goal | Resource Requirement | Estimated Turns |
| :--- | :--- | :--- | :--- |
| **First Harvest** | Complete hay chain | 3+ Hay tiles | 1 turn |
| **Granary Build** | Unlock storage | 150◉ + 10 Wood Planks | 3-5 sessions (30-50 turns) |
| **Kitchen Build** | Unlock expeditions | 400◉ + 20 Wood Planks | 5-7 sessions (50-70 turns) |
| **Quarry Foundry** | Second settlement | 350◉ (Travel + Found) | 5 sessions (50 turns) |

### Total Initial Progression
*   **Total Sessions**: ~20-30 sessions (at 10 turns each).
*   **Total Coins**: ~1000◉ earned (spent on travel, founding, and core buildings).
*   **Total Wood**: ~100 Logs (converted to 30 Planks via upgrade chains).

## 2. Resource Conversion Rates

| Input | Output | Station | Note |
| :--- | :--- | :--- | :--- |
| 5 Logs | 1 Plank | Chain Upgrade | Every 3rd tile in a chain upgrades |
| 5 Grain | 1 Supply | Kitchen | Standard provision for Mine/Harbor |
| 1 Flour + 1 Egg | 1 Bread | Bakery | Food ration (+2 turns) |

## 3. Expedition Efficiency (Quarry)

| Provision | Turn Value | Note |
| :--- | :--- | :--- |
| **Supplies** | 1 turn | Standard entry cost (3 supplies = 3 turns) |
| **Apple** | 1 turn | Base farm drop |
| **Bread** | 2 turns | Bakery craft |
| **Cured Meat** | 3 turns | Kitchen craft |
| **Iron Ration** | 5 turns | High-tier provision |

*Target threshold for standard Mine expedition: 6+ turns (e.g., 3 Supplies + 1 Cured Meat).*

## 4. Almanac XP Sources

*   **Order Fulfillment**: 5 XP
*   **Quest Claim**: 20 - 40 XP
*   **Boss Victory**: 25 XP
*   **Keeper Trial Resolve**: 50 XP

**Level 2 Unlock (Mine capability)**: 150 XP.
*   Required: ~10 orders + 5 quests (achievable within the first 10-15 sessions).
