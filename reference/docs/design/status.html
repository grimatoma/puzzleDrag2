Decision log

  • 2026-06-22 Balance Profile generated; goals + KPI bands captured. Campaign pacing harness + code-derived progression oracle added; the softlock surfaced. The game-design prose set consolidated; the Zone Atlas (10 zones, passes 1–2) built.
  • 2026-06-23 Progression softlock fixed. Home re-costed onto farm goods alone (Bakery off block → hay_bundle, unlocked at Outpost; Village/City off mine goods), so home climbs Outpost→City unaided. Oracle flips to no-softlock.
  • 2026-06-24 game-design split set finalized (the seven-town arc, build-status tracker, meta & money).
  • 2026-06-26 This consolidation. Merged scope + balance + progression into this design/ canon. Decisions: (1) one reconciled roster — 7 required + 5 optional frontier, Thornwild folded; (2) non-linear founding + soft tier/board gates; (3) finale = complete the required spine → build the Monument, retiring the 3-Hearth-Token gate and the act3_win festival-larder beat; (4) home ladder = 4 rungs (Outpost→Town); (5) the no-softlock invariant is canon; (6) one sell price per key. Superseded prose archived (below).
  • 2026-06-26 Zones 1–2 baseline aligned. starting-zones-scope.html confirmed as the most-accurate early-game plan and un-archived as a live companion. Corrected the canon to it: home rung 3 is Town (not City); home + Two Rivers tier-up costs and Two Rivers' 3-tier ladder (Camp 4 → Diggings 7 → Boomtown 10) re-pointed to the baseline's proposed values; founding is a resource caravan, not the coin curve. Plan-vs-code deltas flagged for the later code pass: code names the stone good block (plan: brick) and the home top tier city (plan: Town); founding is still coin-priced in code.
