Implemented economy model live

This section is the actual tuned economy in code (not the PC2 reference below). It was derived from an income-per-run model and a moderate run-budget target. See the Balance & Journey ledger (reference/docs/balance/index.html) for the full writeup.

The income-per-run model

  • Board & run: 6×6 = 36 cells, base min-chain 3, ~10 turns/run (1 turn = 1 chain) → ~10 chains/run, avg chain ~5 → ~50 tiles collected/run.
  • Resource income: a chain of N tiles credits floor(progress / TILES_PER_RESOURCE[tile]) of its produced resource — a home run yields ~2–3 each of hay_bundle / plank / flour (more with bonus-yield buildings, workers, season weighting).
  • Coin income: floor(N · value) per chain — raised ~2× from the old floor(N·value/2) so coins keep pace with the breadth of coin sinks.
  • Decoupled income knob: TILES_PER_RESOURCE (resource yield) is now separate from UPGRADE_THRESHOLDS (board-tile upgrade rate) — tune one without disturbing the other.

The other sinks sit in-range: building costs are resource-only too (a handful of planks/ore/goods each, tier-gated; the per-building Cost to build chip on each building page is the source of truth), tools cost 1–2 resources, and founding a new settlement costs 300 coins and up (SETTLEMENT_FOUNDING_BASE_COINS). Coins are spent mainly on founding and the Market, not on raising buildings or tiers.

The settlement tier ladder

PC2's per-town rank ladder folds onto our live home ladder (the live table above; see also Direction). Workers

Every PC2 worker is a threshold reducer ("1 less grain to make bread", "1 less iron to make an iron bar"). This maps directly to our threshold_reduce / threshold_reduce_category ability, and the worker chain confirms the category upgrade ladders (grass→hay, grain→bread→veg→fruit→flower; stone→block, iron→bar→silver→gold). Tools

PC2 tools are "collect all of type X" / "convert type" consumables priced 20–100 coins (farm) with a per-tool Max and XP. Our tool roster mirrors many ([[tools:rake]] = Rake/collect grass, [[tools:axe]] = Axe/collect trees, [[tools:bomb]] ≈ Explosives, [[tools:drill]] for ore).
