The core loop

Drag a chain of 3+ matching tiles (8-direction, no reuse) → spend one turn → collect resources + coins → refine at buildings → climb the settlement → raise the wagon to the next ruin. The loop below is WIRED end to end; the structure that sequences it across towns is the plan.

The town-tier ladder (Camp → City)

Each ruin climbs the same rank ladder. A tier-up is one town-center upgrade costing escalating local resources that (a) unlocks the next batch of content, (b) raises caps, (c) adds plots. The top tier builds the wagon to the next ruin. This replaces today's scattered lv gates and is PLANNED — costs below are first-pass proposals for the balance simulator.

Unlock gates — what's available when

Building-gated tile spawners (the backbone mechanism)

Beyond the two staple tiles, a category appears on the board when you build the structure that produces it: a Coop spawns birds (eggs), a Lumber Camp spawns oak ([[resources:plank]]), a Garden spawns carrots ([[resources:soup]]). The town you construct reshapes the board you play on.

The forcing function: each tile's resource is consumed by a later cost ([[resources:plank]] gates the [[buildings:mill]]/[[buildings:bakery]] and every tier-up; [[resources:eggs]] gates [[recipes:rec_bread]]; the 12 [[resources:soup]] is part of the City→Wagon cost) — so every spawner is eventually required, but the player picks the order.

Implementation reality: PARTIAL the pool_weight ability mechanism ships and its scope includes "building", but no building uses it yet and there is no Coop/Garden/Lumber-Camp. Wiring it is the headline plan task — PLANNED.

The three bridges (Town 2+)
  • Farm food → mine turns. WIRED — keep the expedition model; make it the hard gate at Town 2 (out of food → session ends).
  • Mine metal → higher tiers + tools. PLANNED — from Town 2, tier-ups + [[buildings:workshop]]/[[buildings:forge]] cost mine resources.
  • Better tools → both boards faster. WIRED — tools are global, shared across settlements.

The economy loop

The pipeline: chains generate raw resources → buildings & recipes refine them → refined goods sell for coins or feed the next cost → coins, tools, workers, level, and knowledge carry globally while resources stay per-settlement.

Carries globally (meta)Stays per-settlement
Coins — bootstrap a fresh ruin via its MarketResources / inventory
Player level + XP + Almanac perksBuildings — each town from scratch
Tools · Knowledge (researched tiles, recipes)Town tier / rebuild progress · board · hazards · biome
Workers (hired once, help everywhere) + runes—

The load-bearing refactor: state.inventory is keyed by settlement id (Record<zoneId, Inventory>); coins, tools, workers, level, and knowledge stay global. Each ruin keeps its own resource pool — travel leaves stock behind and a fresh settlement bootstraps from global coins via its Market. WIRED
